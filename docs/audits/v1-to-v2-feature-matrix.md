## 9. 已确认产品决策

1. 首版不包含成人专用内容、成人源及成人关键词过滤。
2. 搜索历史跨设备同步，默认开启，最多 50 条、保留 90 天，用户可停止记录和清空。
3. 首版不提供 PWA。
4. 首版不提供移动端长按倍速，保留移动端播放器控制锁。
5. 用户导出包含收藏、观看历史、标准化进度、偏好、来源选择、搜索历史和本人提交，不含秘密及媒体 URL。
6. 首版推荐标签由系统提供，不允许用户自定义。
7. 播放失败只推荐候选来源，必须由用户确认后切换。
