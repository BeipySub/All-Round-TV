# V2 产品文档索引

本目录是 All-Round-TV V2 的产品事实源。文档描述“做什么、谁能做、在什么状态下如何反馈”，不替代 API、数据库或视觉设计文档。

| 文档 | 用途 |
| --- | --- |
| [产品需求](product-requirements.md) | 产品目标、首版范围、模块逻辑与验收标准 |
| [业务规则](business-rules.md) | 跨页面不变量、排序、匹配、保留和异常规则 |
| [角色权限](roles-and-permissions.md) | 角色能力、对象权限和数据可见性 |
| [信息架构](information-architecture.md) | 页面树、导航、页面职责与页面状态 |
| [业务状态机](state-machines.md) | 邀请、账户、搜索、审核、播放与迁移状态 |
| [需求追踪矩阵](requirement-traceability.md) | V1 功能到 V2 需求、页面、权限和状态机的映射 |

## 文档优先级

1. 已批准的产品决策与业务规则。
2. 本目录的产品需求、权限和状态机。
3. Figma 已评审原型。
4. API 与技术设计。

若原型或实现与产品规则冲突，应先更新并评审产品文档，不能直接用代码行为反向覆盖产品决策。
