# 观影、推荐、来源选集与广告追踪原型 v0.5

直接在浏览器打开 `index.html`。本原型不请求网络，也不属于生产前端代码。

## v0.5 调整

- 主菜单不再展示搜索文字，仅保留搜索图标。
- 点击搜索图标后，搜索框从导航下方展开；支持关闭、Esc 和 `/` 快捷键。
- 提交快速搜索后进入完整搜索结果页。
- 播放器下方新增配置来源标签和来源实际剧集列表。
- 切换来源会刷新该来源的线路、健康状态和剧集数量。
- 选择剧集后创建新的安全播放会话，并更新当前播放来源与集数。
- 广告标记限定为已注册、已登录且账户有效的用户。
- 广告标记确认框显示当前账户归属，提交后先作为个人标记生效。

## 演示路径

1. 点击顶部搜索图标，体验下滑搜索面板。
2. 进入播放页，在视频下方切换流光影视、白鲸资源或远山影视。
3. 从对应来源剧集列表选择目标集数，观察播放会话连接状态。
4. 点击“标记广告开始→标记广告结束”，确认注册用户标记。
5. 使用实验工具进入可信广告段，体验手动/自动跳过和撤销。

## 产品边界

- 来源必须是管理员启用且用户选择的配置来源。
- 不同来源的剧集数量、线路和命名可以不同。
- 只有可靠映射到同一标准剧集时才能沿用进度。
- 访客、会话失效用户和禁用账户不能提交广告标记。
- 使用已有可信标记跳过广告与创建新标记是两个不同权限。

Figma 仍是最终原型事实源；本目录用于评审交互。
