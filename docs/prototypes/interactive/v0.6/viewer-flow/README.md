# 观影、推荐、来源选集与广告追踪原型 v0.6

直接在浏览器打开 `index.html`。本原型不请求网络，也不属于生产前端代码。

## v0.6 搜索调整

- 主菜单继续只保留搜索图标，不显示“搜索”文字菜单。
- 点击图标打开居中的悬浮搜索模态框，不再从导航下方展开。
- 弹窗提供背景遮罩，并使用原生对话框管理键盘焦点。
- 支持 Esc、关闭按钮和点击弹窗外遮罩关闭。
- `/` 快捷键可在非输入状态下打开搜索。
- 最近搜索点击后只填充输入框，不会未经确认直接发起查询。
- 提交关键词后关闭弹窗并进入完整搜索结果页。
- 移动端弹窗靠近页面顶部展示，并限制最大高度。

## 延续能力

- 播放器下方按配置来源加载线路和剧集。
- 用户选集后创建新的安全播放会话。
- 同类推荐点击先进入详情，不直接播放。
- 广告标记仅限已注册、已登录且账户有效的用户。
- 支持可信广告手动/自动跳过和撤销。

Figma 仍是最终原型事实源；本目录用于评审交互。
